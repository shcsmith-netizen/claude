### 컨셉 스토리

장마 = 하락장. 비를 피하듯 하락을 피하는 매매. 매일 장 마감 후 "오늘 비가 왔는지, 갤지" 정리하고, 우산 챙길 타이밍(리스크 신호)을 같이 짚어주는 블로그.

## 3. 컨텐츠 템플릿

### A. 평일 마감 브리핑 (16:30 발행)

**제목 형식**
- `코스피 마감 [+0.85%] 외인 8천억 순매수, 반도체 주도`
- `[2026년 4월 29일] 코스피 마감 정리 | 장마여`
- 메인 키워드 **"코스피 마감"** 제목 앞쪽 배치

**구조 (AI 친화 — 결론 우선 + 카드뉴스 비주얼)**
```
[썸네일 카드뉴스: 지수+등락률+한 줄 요약, 라이트 톤]

▶ 핵심 요약 박스 (3줄)
- 코스피 2,XXX (+0.85%), 코스닥 8XX (+0.42%)
- 외인 +8,200억 / 기관 -3,100억 / 개인 -5,100억
- 주도 섹터: 반도체·2차전지 / 부진: 바이오·건설

▶ 오늘 시장 한 줄 (감성)
"장마 그친 오후처럼 반등 마감"

[소제목 1] 코스피 마감 — 오늘 지수 어땠나
[이미지: KOSPI/KOSDAQ 미니 차트 카드 (이미지 3 상단 스타일)]
- 본문 (성현님 평소 말투)

[소제목 2] 수급 — 외인·기관·개인 누가 샀나
[이미지: 수급 카드 (3분할 박스, 화살표 포함)]
- 표로 정리

[소제목 3] 주도 테마 — 오늘 시장을 끌어올린 섹터
[이미지: 주도테마 마인드맵 (이미지 3 스타일, 자동 생성)]
- 섹터별 핵심 종목 1~2개씩

[소제목 4] 섹터 트리맵 — 한눈에 보는 업종별 등락
[이미지: 트리맵 (이미지 4 스타일, plotly 자동 생성)]
- 진한 빨강 = 강세, 진한 파랑 = 약세

[소제목 5] 내일 체크포인트 — 우산 챙길까?
[이미지: 캘린더 카드]
- 다음 거래일 일정/이벤트

▶ FAQ (AI 검색 최적화)
Q. 오늘 코스피 왜 올랐나요?
Q. 외국인이 가장 많이 산 종목은?

▶ 마무리 코멘트 (개인 톤)
면책: 본 글은 정보 제공 목적이며 투자 권유가 아닙니다.

#태그 15~20개
```

## 4. 글 디자인 가이드 (예쁘게 쓰기)

### 4-1. 시각 요소

| 요소 | 사용법 |
|---|---|
| **볼드** | 핵심 수치(지수, 등락률, 매매대금) — 한 단락에 1~2개만 |
| 색상 강조 | 상승=빨강(#E03131), 하락=파랑(#1971C2), 강조=검정 볼드. 형광색·노랑 금지 |
| 표 | 지수/수급/섹터별 등락률은 무조건 표로 |
| 인용구 | "오늘 시장 한 줄" 감성 코멘트만 인용구 처리 |
| 구분선 | 섹션 간 ━━━ 구분선으로 호흡 |

### 4-2. 이미지 통일성 (브랜딩) — **라이트 톤 + 카드뉴스 스타일**

> **톤 결정 의견:** 컨셉이 "장마(하락) **피하기**"라 비 내리는 어두운 이미지보다 **비 갠 후 맑은 하늘**이 더 컨셉과 맞아요. 어두운 톤은 위협·공포 느낌이 강해서 매일 보기 부담스럽고, 밝은 톤은 "장마 지나고 여기서 정리하자"는 안정감을 줍니다. 경제 인플루언서 상위권도 대부분 라이트 톤(화이트·베이지)이라 인플루언서 탭 정렬에서도 자연스럽게 어울립니다.

> **레퍼런스 분석 (성현님 제공 4종):**
> - **이미지 1 — 특징주 마인드맵형**: 중앙 "주도테마" → 분기로 섹터별 종목·등락률 노트 필기 느낌. 친근하고 정보 밀도 높음
> - **이미지 2 — 카드뉴스 6분할 그리드**: 각 셀에 아이콘+제목+짧은 설명. 한눈에 스캔 가능
> - **이미지 3 — 주식테마맵 (FINUP 스타일)**: 마인드맵 + 차트 + 거래대금을 한 장에 통합. 가장 완성도 높음
> - **이미지 4 — 섹터 트리맵**: 색상 진하기로 등락률 표현. 데이터 시각화 끝판
>
> → **3가지 스타일을 본문 자동 생성에 모두 도입**. 매일 같은 패턴이면 지루하니 **요일별 메인 비주얼 로테이션**.

#### 콘텐츠별 메인 비주얼 (자동 생성)

| 종류 | 비주얼 | 생성 방식 |
|---|---|---|
| **썸네일** | 카드뉴스 (지수/등락률/한 줄 요약) | PIL + 템플릿 |
| **지수 마감 카드** | KOSPI/KOSDAQ 미니 차트 + 수급 | matplotlib |
| **주도테마 맵** | 마인드맵 (이미지 3 스타일) | graphviz/networkx |
| **섹터 트리맵** | 등락률 트리맵 (이미지 4 스타일) | plotly/squarify |
| **이슈 카드뉴스** | 6분할 그리드 (이미지 2 스타일) | PIL 카드 합성 |

## 5. AI 친화 SEO 전략 (AI 검색 노출 최적화)

### 5-1. 핵심 원칙: **결론 우선 (Inverted Pyramid)**

AI는 글 상단에서 답을 찾는다. 첫 100~200자 안에 **누가/언제/무엇을/얼마/왜** 다 들어가야 함.

```
[첫 문단 = AI가 인용할 영역]
2026년 4월 29일 코스피는 전일 대비 0.85% 오른 2,712.34로 마감했습니다.
외국인이 8,200억 순매수하며 반도체 업종을 끌어올렸고, 코스닥도 0.42% 상승했습니다.

[그 다음에 상세 분석]
```

### 5-2. AI가 좋아하는 글 구조

1. **TL;DR / 핵심 요약 박스** — 3줄 이내, 글 맨 위
2. **질문형 소제목** — "코스피 마감 어땠나?", "외인은 왜 샀을까?"
3. **숫자 명시** — "큰 폭 상승" ❌ → "0.85% 상승, 23p 오른 2,712" ✅
4. **표/리스트 적극 활용** — AI가 구조 파싱하기 쉬움
5. **고유명사 정확히** — 종목명, 지수명, 날짜 풀네임
6. **FAQ 섹션** — "오늘 코스피 왜 올랐나요?" 같은 자문자답 1~2개

### 5-3. 메인 키워드 운영

- **상시 메인 키워드**: `코스피 마감`, `코스닥 마감`, `오늘 증시`, `외국인 매매동향`
- **일자별 키워드**: `[YYYY년 MM월 DD일] 코스피 마감`
- **롱테일**: `코스피 종가`, `반도체 종목 마감`, `[종목명] 종가`
- 본문 5~8회 자연 반복, 소제목마다 1회 포함
